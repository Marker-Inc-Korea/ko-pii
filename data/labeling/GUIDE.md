# 공공문서 인간 라벨링 가이드 — 가중치 공개 게이트

**목적**: 튜닝 NER 의 실제 공공문서 일반화 성능을 *사람 라벨* 기준으로 측정 —
[`docs/HYBRID_NER.md`](../../docs/HYBRID_NER.md) "모델 공개 (예정)" 절의 마지막 선행 조건.
예상 소요: 100문서 × 2~4분 ≈ **반나절**.

## 작업 방법

1. `labeling_sample.jsonl` 의 각 줄(문서)을 열어 `spans` 를 교정한다.
   prefill 은 ko-pii 룰 출력일 뿐 — **틀린 것 삭제, 빠진 것 추가, 경계 수정**.
2. 교정이 끝난 문서는 `"reviewed": true` 로 바꾼다.
3. 끝나면 채점: `python data/labeling/score_labeled.py` (룰 F1)
   — ML 채점은 `KO_PII_NER_MODEL=<모델경로>` 지정 시 함께 출력.

## span 규칙

- **조사 제외**: "홍길동이 제출한" → `홍길동` 만 (이 제외)
- **PERSON 은 한글 2자+ 실명만**: 1음절 성씨("김"), "김모씨", "○○○" 등 가명 표기는 제외
- **공인의 직무 수행 정보도 표기** (장관·과장 실명 등) — 모델은 검출하고, 마스킹 여부는
  정책 계층의 일이므로 라벨 단계에서는 사람 이름이면 전부 PERSON
- **ID 류**(주민·전화·사업자 등)는 구분자 포함 전체 문자열
- 판단이 애매하면 `"note"` 필드에 메모를 남기고 일단 표기 (검토 시 일괄 판정)
