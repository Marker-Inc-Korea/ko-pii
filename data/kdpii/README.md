# KDPII — 출처 및 라이선스 고지

본 디렉토리의 `train/valid/test.json` 은 **KDPII**(Korean Dialogic Dataset for
Personal Identifiable Information De-identification)의 **원본 무수정 재배포**입니다.

- **원 배포처**: Zenodo — https://zenodo.org/records/10968609
- **라이선스**: **CC-BY-4.0** (Creative Commons Attribution 4.0 International)
- **저작자**: Fei et al. (연세대학교 외), *KDPII: A New Korean Dialogic Dataset for
  the Deidentification of Personally Identifiable Information*, IEEE Access, 2024.
  DOI: [10.1109/ACCESS.2024.3461804](https://doi.org/10.1109/ACCESS.2024.3461804)
- **변경 사항**: 없음 (원본 그대로). ko-pii 는 본 데이터를 평가(`ko_pii.eval.kdpii`) 및
  하이브리드 NER 학습 레시피(`docs/HYBRID_NER.md`)에 사용합니다.

인용 BibTeX 은 [`docs/EVALUATION_REPORT.md`](../../docs/EVALUATION_REPORT.md) 하단 참조.
